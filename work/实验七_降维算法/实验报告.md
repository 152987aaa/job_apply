## 二、实验内容与方法

### 2.1 PCA算法原理

PCA（主成分分析）是基于方差最大化的线性降维方法：

**核心步骤：**
1. **数据去中心化**：对每维特征减去均值
2. **计算协方差矩阵**：$C = \frac{1}{n-1} X^T X$
3. **特征值分解**：求解协方差矩阵的特征值和特征向量
4. **主成分选择**：按特征值从大到小排序，选取前k个
5. **数据投影**：$Y = X W$

### 2.2 SVD算法原理

奇异值分解对任意矩阵$X$分解为：
$$X = U \Sigma V^T$$

**核心步骤：**
1. 计算$X^T X$的特征值与特征向量得到$V$
2. 计算$X X^T$的特征值与特征向量得到$U$
3. 奇异值$\sigma_i = \sqrt{\lambda_i}$
4. 截断SVD：保留前k个最大奇异值

## 五、实验对比分析

### 5.1 算法对比

| 对比维度 | PCA | SVD |
|----------|-----|-----|
| 数学原理 | 协方差矩阵特征值分解 | 奇异值分解 |
| 计算效率 | 高 | 较高 |
| 信息保留 | 高 | 高 |
| 适用场景 | 数据压缩、去噪、可视化 | 推荐系统、NLP、图像 |
| 是否需要中心化 | 是 | 否 |
| 核心指标 | 方差解释率 | 奇异值贡献率 |

### 5.2 结果分析

1. **PCA降维**：保留95%方差需要28维，说明数据冗余度较低
2. **SVD降维**：保留95%奇异值贡献需要29维，与PCA结果相近
3. **自定义实现与sklearn对比**：结果一致，验证了算法正确性

## 六、实验总结

1. 成功实现了PCA和SVD两种降维算法
2. PCA基于协方差矩阵特征值分解，需要数据中心化
3. SVD基于奇异值分解，不要求数据中心化，适用范围更广
4. 两种算法在音乐数据集上表现相近，SVD略优
